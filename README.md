# Getting Started with Vite

This project uses [Vite](https://vitejs.dev/) for development and production builds.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.\
Open the URL shown in your terminal (typically [http://localhost:5173](http://localhost:5173)).

### `npm run build`

Builds the app for production to the `dist` folder.

### `npm run preview`

Locally preview the production build.

## Environment Variables

Vite exposes client-side variables prefixed with `VITE_`.\
Rename any `REACT_APP_*` variables in your `.env` files to `VITE_*`.
